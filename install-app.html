Pixel to EM

Font Converter Offline Web Application

[PWA Experiment]

Pixel to EM is an Offline Web Application, which is used to calculate (convert) values from one source to another. For example: convert a pixel value to in points, or percent to in pixel.
Below is a tutorial on how to install the app.

Simple Tutorial

Access the website at your smartphone

1 — Enter em https://pixel.fcschmidt.me
2 — Go to Customize and control Google Chrome

Tab Customize and control Google Chrome

3 — Click Add to Home screen to install the app.

Add to Home screen

Click ADD to confirmation the installation in your Home screen.

Home Screen

App installed on the home screen.
Click the icon to access the app.
