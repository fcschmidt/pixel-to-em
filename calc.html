Calculation Method

Conversions based on 16px browser default size.
16px is used as text size of the body in all conversions. This is the default value browser. You can change this value for the size you used as the basis.

  • PX to EM:
    Formula: input value / pixel base.
    Exemple: 16px / 16px = 1em
  • PX to Percent %:
    Formula: input value / pixel base * 100.
    Exemple: 16px / 16px * 100 = 100%
  • PX to Points:
    Formula: input value * (dots per inch / pixel per inch)
    Exemple: 16px * (72px/96px) = 12pt
  • EM to PX:
    Formula: input value * pixel base
    Exemple: 1em * 16px = 16px
  • EM to Percent %:
    Formula: input value * 100
    Exemple: 1em * 100 = 100%
